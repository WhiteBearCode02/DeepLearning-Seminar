# 🎓 Easy! Deep Learning 세미나 아카이브
> **순천향대학교 소프트웨어공학과 이현준**
> "기초부터 심화까지, 추론을 통해 배우는 딥러닝 핵심 이론"

이 저장소는 딥러닝의 기초 개념부터 심층 신경망의 고질적인 문제들을 해결해 나가는 과정을 파트별로 기록한 세미나 저장소입니다. 단순히 지식을 나열하는 것이 아니라, **'왜 이런 문제가 발생했고, 어떻게 해결하는가'**에 대한 공학적 추론 과정을 담고 있습니다.

---

## 🗺 세미나 로드맵

본 저장소는 단계별 학습을 위해 다음과 같이 구성되어 있습니다.

### 📂 [Part 1. AI 패러다임의 이해](./part1/)
- **핵심 주제**: AI/ML/DL의 정의 및 규칙 기반에서 데이터 기반으로의 변화
- **주요 내용**: 인공신경망(DNN)의 구조적 이해와 학습 방식의 차이점 분석
- **발표 자료**: `Easy!딥러닝1.1~1.2`

### 📂 [Part 2. 역전파와 활성화 함수](./part2/)
- **핵심 주제**: 기울기 소실(Vanishing Gradient) 문제와 ReLU의 혁신
- **주요 내용**: 오차 역전파 메커니즘(액나/액웨), 시그모이드의 한계 극복, 모델 평가 지표(F1-Score, AUC)
- **발표 자료**: `Easy!딥러닝6.1.1~6.1.3`

## 💡 이 저장소의 특징
1. **추론 중심 학습**: 수식의 나열보다는 현상의 원인과 해결책 사이의 논리적 연결을 중시합니다.
2. **시각적 분석**: 시그모이드와 ReLU의 실험 결과 비교를 통해 활성화 함수의 중요성을 시각화했습니다.
3. **평가 다각화**: 단순 정확도를 넘어 정밀도와 재현율을 통한 정교한 모델 평가 방식을 다룹니다.
