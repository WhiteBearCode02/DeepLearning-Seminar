# 🧠 Easy! 딥러닝 세미나 - Part 1

본 세션에서는 인공지능(AI), 머신러닝(ML), 딥러닝(DL)의 정의와 상호 관계, 그리고 AI 학습 방식의 패러다임 변화를 다룹니다.

---

## 01. AI, ML, DL의 개념 및 관계

### 1. 정의 (Definitions)
- **AI (Artificial Intelligence)**: 인간의 지능을 인공적으로 구현한 기술 전체를 아우르는 광범위한 개념입니다.
- **ML (Machine Learning)**: 데이터를 통해 컴퓨터를 스스로 학습시키는 기술입니다. 사람이 데이터의 특징(Feature)을 가이드해주는 방식이 포함됩니다.
- **DL (Deep Learning)**: 인간의 뇌 구조를 모방한 **인공신경망(DNN)**을 사용하는 기술입니다. 데이터에서 무엇이 중요한지 인공신경망이 스스로 찾아내어 깊게 학습합니다.

### 2. 기술 간의 포함 관계
AI는 가장 큰 범주이며, 그 안에 ML이, 그리고 ML의 한 분야로 DL이 존재합니다.
- **AI**: 규칙 기반 알고리즘
- **ML**: 선형 회귀, 퍼셉트론 등
- **DL**: 심층 신경망(Deep Neural Network) 활용



---

## 02. AI 학습 방식의 변화

AI가 문제를 해결하는 방식은 과거에서 현재로 오며 큰 패러다임의 변화를 겪었습니다.

| 구분 | 규칙 기반 방식 (Rule-based) | 데이터 기반 방식 (Data-driven) |
| :--- | :--- | :--- |
| **특징** | 사람이 모든 규칙을 직접 프로그래밍 | 데이터로부터 컴퓨터가 스스로 규칙을 학습 |
| **장점** | 결과에 대한 원인 파악이 명확함 | 복잡하고 비정형적인 데이터 처리에 강함 |
| **한계** | 예외 상황이 많을 경우 구현이 어려움 | 학습을 위한 양질의 대량 데이터가 필요함 |
