# 📂 Part 1. AI, ML, DL의 개념 및 패러다임 이해

이 파트에서는 인공지능의 가장 기초가 되는 정의와 기술 간의 상관관계, 그리고 현대 AI가 데이터를 처리하는 방식의 근본적인 변화를 다룹니다.

## 📝 학습 목차
1. **AI, ML, DL의 정의**: 각 기술이 수학과 방정식, 고차 방정식으로 진화하는 과정.
2. **기술 간의 포함 관계**: AI > ML > DL의 구조적 이해.
3. **학습 패러다임의 변화**:
    - **규칙 기반(Rule-based)**: 인간이 정한 규칙의 한계.
    - **데이터 기반(Data-driven)**: 기계가 스스로 규칙을 찾는 혁신.

## 📊 핵심 포인트
- **ML**은 사람이 가이드하는 방식인 반면, **DL**은 인공신경망이 스스로 중요 특징을 추출합니다.
- 딥러닝의 핵심인 **DNN**은 은닉층을 2개 이상 지닌 학습 방법을 의미합니다.
