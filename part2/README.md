# 📂 Part 2. ReLU 활성화 함수와 역전파의 심화 이해

심층 신경망 학습의 최대 난제인 '기울기 소실' 문제를 수학적으로 분석하고, 이를 해결하기 위한 ReLU 활성화 함수의 메커니즘을 탐구합니다.

## 📝 학습 목차
1. **기울기 소실(Vanishing Gradient)**: 역전파 시 기울기가 0에 수렴하여 앞쪽 층이 학습되지 않는 현상.
2. **활성화 함수 비교**: Sigmoid의 한계($0.25$ 미분값)와 ReLU의 혁신(양수 미분값 $1$).
3. **오차 역전파의 흐름**: '액나(Output Error)'가 '액웨(Hidden Error)'로 전달되는 과정 분석.
4. **ReLU의 변형**: Dying ReLU 문제를 해결하기 위한 Leaky ReLU 및 PReLU 학습.

## 📊 실험 결과 요약
- Sigmoid 대비 ReLU 사용 시 **Loss 하락 속도**가 비약적으로 상승함을 확인.
- 모델이 너무 단순할 때 발생하는 **과소적합(Underfitting)** 문제를 해결하기 위한 구조적 접근법 정리.
